# 🛡️ Level Up – Internet Segura

Um jogo educativo desenvolvido na plataforma **Roblox** com o objetivo de conscientizar crianças e adolescentes sobre os perigos da internet e da falsidade digital.

## 📖 Sobre o Projeto

**Level Up – Internet Segura** é um jogo inovador que combina diversão e educação. O jogador assume o papel de um **Guardião Digital** que deve recuperar seu computador sequestrado por um hacker, atravessando desafios no estilo **parkour (obby)** e respondendo questões sobre segurança digital.

### 🎮 Características Principais

- **Gameplay Interativo**: Desafios de parkour combinados com perguntas sobre segurança
- **Ameaças Reais**: Enfrente situações inspiradas em golpes reais (phishing, links falsos, etc.)
- **Aprendizado Divertido**: Respostas erradas geram explicações educativas
- **Progresso Estruturado**: Sistemas de checkpoints e portais para melhor experiência
- **Linguagem Luau**: Desenvolvido com Roblox Studio

## 🎯 Objetivo

Ensinar conceitos de segurança digital de forma divertida, acessível e envolvente, incentivando os jogadores a:
- ✅ Reconhecer ameaças online
- ✅ Proteger suas contas
- ✅ Navegar na internet com segurança
- ✅ Compreender riscos de engenharia social

## 🚀 Como Usar Este Repositório

Este repositório contém o site de apresentação do projeto "Level Up – Internet Segura".

### 📁 Estrutura de Arquivos

```
LevelUpInternetSegura/
├── index.html          # Página principal do site
├── styles.css          # Estilos e design
├── script.js           # Funcionalidades interativas
└── README.md           # Este arquivo
```

### 🌐 Acessar o Site

1. **Com GitHub Pages**:
   - Vá para Settings → Pages
   - Escolha "Deploy from a branch"
   - Selecione `main` ou `master` como branch
   - Seu site estará disponível em: `https://Sobreira652.github.io/LevelUpInternetSegura`

2. **Localmente**:
   - Baixe os arquivos
   - Abra `index.html` no seu navegador

## 📝 Como Adicionar Conteúdo

### Adicionar Screenshots

1. Coloque suas imagens na pasta `assets/images/`
2. Edite `index.html` e substitua os placeholders:

```html
<div class="gallery-item">
    <img src="assets/images/screenshot1.jpg" alt="Screenshot 1">
</div>
```

### Adicionar Vídeos

1. Copie o link de incorporação (iframe) do YouTube ou Vimeo
2. Substitua os placeholders na seção de vídeos:

```html
<div class="video-item">
    <iframe src="https://www.youtube.com/embed/VIDEO_ID" allowfullscreen></iframe>
</div>
```

### Atualizar Link do Jogo

Na seção "Jogar Agora", substitua o link padrão pelo link real do seu jogo no Roblox:

```html
<a href="https://www.roblox.com/games/SEU_GAME_ID" target="_blank" class="btn btn-large">
    Jogar no Roblox
</a>
```

## 🎨 Personalizações

### Cores Padrão

O site utiliza um tema dark moderno com cores vibrantes:
- **Cor Primária**: `#FF6B35` (Laranja)
- **Cor Secundária**: `#004E89` (Azul Escuro)
- **Cor de Destaque**: `#1AC8ED` (Ciano)
- **Fundo**: Degradê azul escuro

Para mudar as cores, edite as variáveis em `styles.css`:

```css
:root {
    --primary-color: #FF6B35;
    --secondary-color: #004E89;
    --accent-color: #1AC8ED;
    /* ... */
}
```
